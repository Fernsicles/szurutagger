# szurutagger
A bot to tag szurubooru posts using deepdanbooru

# Usage
Deepdanbooru needs to be installed.
Put the username, password, and url for the bot to use in `config.json`
```json
{
	"user": "szurutagger",
	"password": "password",
	"url": "https://example.com"
}
```
Then run `python szurutagger.py`
